Simple Setup app will allow user to control their set top box using the TV remote control (via IR control to the set top box).

 

Setup via App for connected set top box

  1. Connect a set top box via HDMI to TV.
  2. Launch Simple Setup app on TV
  3. To setup for a new set top box, select the brand and model for your set top box.

 

Control of set top box using TV remote control

  1. When user is watching TV source, TV remote control will be in TV mode and sending TV IR codes.
  2. When Simple Setup app is running on TV, it will detect when user switch to HDMI input for set top box and trigger TV remote control to switch to send set top box IR codes during key press.
