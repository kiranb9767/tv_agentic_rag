The feature of Hidden keys

Hidden keys will be appeared to extend more control functions while pressing the 123 key. The 123 key is on middle of remote control, which is used to turn on/off hidden keys mode. When hidden key mode is turned on, the number keys 0 to 9 and media play control keys will appear with backlight and trigger the functions. When hidden key mode is turned off by pressing 123 key again or automatically turned off, the remote control will work with normal keys.
