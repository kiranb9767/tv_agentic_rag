Select a Viewing mode

Select your preferred viewing mode for your HDMI source. Go to  (Home) > Settings > Picture > Viewing mode to set the Viewing mode.

The available Viewing modes are:

  • Auto – TV will automatically detect the contents and set the viewing mode accordingly.. 
  • Video – When Viewing mode is manually or automatically set to Video, the Settings > Picture > Picture style/AI Picture style options will be available to select.
  • Game – When Viewing mode is manually or automatically set to Game, the Settings > Picture > Game profiles options will be available to select.
  • Monitor – Set to Monitor mode for computer application display.
