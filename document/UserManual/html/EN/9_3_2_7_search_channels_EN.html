TV / EXIT > OK > OPTIONS > Search channel

You can search for a channel to find it in a long list of channels. The search will return the matching channel name with the result arranged in alphabetical order.

  1. Press TV / EXIT, to switch to TV.
  2. Press  Channel list to open the channel list.
  3. Press OPTIONS > Search channel, and press OK to open on-screen keyboard.
  4. Use navigation keys to select a character, then press OK.
  5. Select Enter when complete the input.
  6. Press  Back to close the channel list.
