Supported Computer/Video resolution

Resolution — Refresh rate

  • 640 x 480 - 60Hz
  • 576p - 50Hz 
  • 720p - 50Hz, 60Hz
  • 1920 x 1080p - 24Hz, 25Hz, 30Hz, 50Hz, 60Hz, 100Hz, 120Hz
  • 2560 x 1440p - 60Hz, 120Hz
  • 3840 x 2160p - 24Hz, 25Hz, 30Hz, 50Hz, 60Hz, 100Hz, 120Hz

* Note: 100Hz is only supported when HDMI Ultra HD is set to Optimal; 120Hz is only supported when HDMI Ultra HD is set to Optimal or Optimal (Auto Game 120Hz Pro).

 

Supported Video only resolution

Resolution — Refresh rate

  • 480i - 60Hz (when HDMI Ultra HD is set to Standard)
  • 576i - 50Hz (when HDMI Ultra HD is set to Standard)
  • 1080i - 50Hz, 60Hz
  • Variable refresh rate video 1080p, 1440p, 2160p with frame rate up to 40-120Hz when HDMI Ultra HD is set to Optimal (Auto Game 120Hz Pro)

* Note: Some resolution and frame rate may not be supported in all input sources.
